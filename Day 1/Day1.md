# What is Deep Learning?

Deep Learning (DL) is a subset of Machine Learning (ML) that uses **Artificial Neural Networks (ANNs)** with multiple layers (hence "deep") to learn patterns from data.

It is inspired by the structure and function of the **human brain**, where neurons are connected and signals are passed between them.

Deep learning models automatically learn:
- Features
- Representations
- Patterns  
directly from raw data.

Unlike traditional machine learning, deep learning does **not require manual feature engineering** for most tasks.

---

## Why is it Called "Deep"?

“Deep” refers to the **number of layers** in a neural network.

Example:
- Input Layer
- Hidden Layer 1
- Hidden Layer 2
- Hidden Layer 3
- Output Layer

More hidden layers → deeper network → more complex pattern learning.

---

# Machine Learning vs Deep Learning

| Aspect | Machine Learning (ML) | Deep Learning (DL) |
|---------|------------------------|--------------------|
| Definition | Subset of AI that learns patterns from data | Subset of ML using multi-layer neural networks |
| Feature Engineering | Manual | Automatic |
| Data Requirement | Works with small to medium datasets | Requires large datasets |
| Performance | Good for structured data | Excellent for unstructured data |
| Hardware | Can run on CPU | Often requires GPU/TPU |
| Interpretability | More interpretable | Less interpretable (“black box”) |
| Training Time | Faster | Slower |
| Examples | Linear Regression, SVM, Decision Trees | CNN, RNN, Transformers |

---

# When to Use ML vs DL?

### Use Machine Learning when:
- Dataset is small
- Data is structured (Excel, SQL tables)
- Interpretability is important
- Limited computational resources

### Use Deep Learning when:
- Dataset is large
- Working with images, audio, video, text
- High accuracy is required
- You have access to GPUs

# Real-World Examples

- Fraud detection

### Deep Learning
- Face recognition
- Self-driving cars
- ChatGPT
- Voice assistants
- Medical image diagnosis

---

# Simple Intuition

Machine Learning:
> "Tell me what features to look at, and I’ll learn the pattern."

Deep Learning:
> "Give me raw data, and I’ll figure out the important features myself."

---

# Key Takeaways – Day 1

- Deep Learning is a subset of Machine Learning.
- It uses multi-layer neural networks.
- It automatically learns features.
- It performs extremely well on unstructured data.
- It requires more data and computational power.
